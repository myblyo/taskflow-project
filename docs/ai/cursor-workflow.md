# Cursor Workflow

## Introducción

Este documento describe cómo se ha utilizado **Cursor** durante el desarrollo del proyecto TaskFlow. Aquí se recopilan los flujos de trabajo, atajos de teclado utilizados y ejemplos concretos donde la inteligencia artificial ha ayudado a mejorar el código.

## Contenido del documento

En este archivo se incluirán:

* Los **atajos de teclado más utilizados** durante el trabajo con Cursor.
* Ejemplos prácticos del uso del **chat contextual** para entender partes del código.
* Casos donde el **autocompletado con IA** ha ayudado a generar funciones o mejorar la lógica.
* Ejemplos donde **Composer** ha realizado cambios en múltiples archivos del proyecto.

## Objetivo

El objetivo de este documento es registrar cómo las herramientas de inteligencia artificial integradas en Cursor pueden mejorar la productividad, facilitar la comprensión del código y acelerar el desarrollo del proyecto.
